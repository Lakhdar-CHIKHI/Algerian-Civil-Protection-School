# National School of Firefighters and Algerian Civil Protection - Official Website

This repository contains the source code for the official website of the National School of Firefighters and Algerian Civil Protection. The website is designed to provide information about recruitment, training programs, and other relevant activities. It serves as a portal for new candidates, current trainees, and international partners.

## Overview

This project is built using Next.js, a React framework that enables server-side rendering and static site generation for optimized performance. It also leverages modern development tools and methodologies to ensure a scalable, maintainable codebase.

## Key Features

- **Responsive Design:** Ensures usability across devices of different sizes.
- **Multi-language Support:** Available in multiple languages for local and international users.
- **SEO Optimized:** Follows best practices for search engine optimization.
- **Accessible:** Adheres to web accessibility standards to provide an inclusive experience.

## Installation
- **Prerequisites:**
    1. Node.js (v14.x or higher)
    2. npm (v6.x or higher) or Yarn (v1.x or higher)
    3. Git (v2.x or higher)
- **Steps to Install:**
    - **Clone the repository:**

        `git clone https://github.com/your-username/firefighter-school-website.git`
    - **Navigate to the project directory:**

        `git clone https://github.com/your-username/firefighter-school-website.git`
    - **Install dependencies:**

        `git clone https://github.com/your-username/firefighter-school-website.git`

- **Usage:**
    - **Running the Development Server:**

        `npm run dev or yarn dev`

    - **Building for Production:**

        `npm run build or yarn build`

    - **Starting the Production Server:**

        `npm run start or yarn start`
    - **Linting and Formatting:**

        `npm run lint or yarn lint`
